✒Git Commands Info:

  1. git --version => Will show the version of the git
  2. git init => To initialze the git in the project
  3. git status => means, it will show what is tracked and what is untracked
  4. git add (file path name)=> Example:git add gitandgithub\intro.html
  5. git add .=> add . this will add all the files
  6. git commit -m "any thing we can write here" m => means message
  7. git log=> it will show who has done what and it will show the commit messages
  8. git reset --hard 3f0ec6db2314e43d3d1a08fe786bbec4ffe35c22 => hard means everything it will clear
  9. git reset --soft 629152322a05bea0eb980a385750fe8019cf8fb8 => u can go to previous commits without deleting the present code by using soft
  10. git clone https://github.com/abc/GitandGitCommands.git => git clone will clone all files in u r file explorer
  11. git pull => means it will pull all the changes from the github
  12. git push => it will push all the commits to the github
